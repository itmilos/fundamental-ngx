Directives Component

Input properties

  • width - specifies the width of the skeleton
    • any string value (will be set as inline styles)
    • 'rand' to set width randomly 30-90%
  • height - specifies the height of the skeleton
    • any string value (will be set as inline styles)
  • type - specifies the shape of the skeleton
    • 'rectangle' to make skeleton rectangle (default)
    • 'circle' to make skeleton circle, one dimension is enough when set (width or height)
    • 'text' to set height accordingly to the font-size, height not used when set
  • animation - whether to play animation
    • boolean

Notice: by default skeleton component inherits its size from a content inside of it so the input properties could be omitted.
